GLASSBOOK · LOCAL-FIRST SYNTHESIS v2

综合处理方案 & 核心交互图

基于「零云端 · iCloud 同步 · BYO 大模型」三大原则重构。AI / 开发者 / 育儿 / 财务健康四类需求分层落位，给出可执行的三阶段路线图，附 6 张可直接指导设计开发的交互图。

✓ 无自有云端 ✓ iCloud 数据同步 ✓ 自定义大模型接入（BYO）
01 决策矩阵 14 个子功能逐一处理。新约束：不做自有云端（无服务器、无用户账号、无云数据库）。数据同步走 iCloud；AI 能力走 on-device 或用户自带大模型。
功能 版本 决策 处理方案 关键理由
① 深度 AI 与自动化（Vibe Coding 风格）
MCP 记账插件（Mac/Desktop） V1.5 采纳 · 差异化 官方 MCP Server 跑在用户 Mac 本地，直接读写 iCloud Drive 上的 Glassbook 数据库（受 App Group 权限）。Claude Desktop、Cline、Zed 开箱即用。不涉及任何远程服务器。 不违反「零云端」约束 —— 所有进程都在用户自己机器上跑，通过 iCloud 与手机数据一致。
语义自动打标（Apple Intelligence / BYO） V1.0 采纳 默认用 Apple Intelligence on-device 模型；Pro 用户可配置 自定义 LLM 端点（OpenAI / Claude / Gemini / Ollama / LM Studio 任选），用自己的 API Key 走自己的网关。 on-device 保隐私保默认可用；BYO 满足极客对速度、模型质量、本地部署 Ollama 的需求。
截图/短信静默识别 V1.0 采纳 iOS：Shortcut 自动化 + Live Activity 悬浮确认条；Android：Notification Listener。看不到 App 也能完成入账。 把「手动记账」降到 0 次点击，这是留存率的关键阈值。
② 开发者与极客专属
订阅制服务管理 V1.0 采纳 · MVP 核心 独立 Tab 或一级页面。支持 VPS / AI 订阅 / 域名 / 流媒体四大预设模板。多币种自动换算（汇率每小时更新）。续费日 T-3 / T-1 / 当日三级提醒。 技术消费者最痛的场景，对标 Bobby / Subby 但深度更深，中文记账 App 尚无认真做的。
汇率自动换算 V1.0 采纳 与订阅功能绑定，首页显示「¥ + 折算后」。数据源：exchangerate-api.com。 订阅场景必需，单独拉出做没意义。
自定义大模型接入（BYO LLM） V1.5 采纳 · 新增 设置页新增「AI 引擎」分区。支持 OpenAI API、Claude API、Gemini API、本地 Ollama、LM Studio 五种预设，也支持自定义 OpenAI 兼容端点（baseURL + apiKey）。驱动打标、年度回顾、AI 洞察、问账。 零云端约束下的最优解：用户用自己的额度、自己的模型、自己的数据主权。
Webhook 事件推送 V1.5 采纳 · 改为设备直出 手机 App 在预算超支 / 订阅到期 / 大额消费时，直接从设备向用户配置的 URL（Slack Webhook、飞书群机器人、n8n、自建 HTTP 接口）发送 POST。无中转服务器。 不需要云端也能实现 —— 只要手机联网时事件触发，设备自己是 HTTP Client。
Open REST API（对外） — 删除 原方案需要云端承载；与「零云端」冲突，砍掉。开放性改由 MCP + Webhook 承担（入站对话 / 出站事件双通道）。 MCP 已覆盖「外部程序查询数据」的需求，且隐私边界更清晰。
私有化部署（Docker Self-Host） — 删除 原方案需要 PostgreSQL 后端，与「零云端」冲突。替代方案：iCloud 已提供跨设备数据主权，且用户不需要会 Docker。 Local-First + iCloud 的组合已满足「数据我自己掌控」这一原始诉求。
本地数据库加密（Argon2id + AES-256） V1.0 采纳 SQLCipher 本地加密；密钥由 Face ID 解锁，不上云。 零额外开发成本（库成熟），但营销卖点强。
③ 家庭与育儿场景
神兽专项开支维度 V1.0 采纳 分类体系里加「孩子」一级分类（下含教育 / 兴趣班 / 玩具 / 医疗 / 日用）。年度报告单独展示。 实现成本极低，但精准命中家庭用户，可做 App Store 截图主打。
家庭共享 + 分级隐私 V1.5 采纳 · 走 CKShare 共享账本基于 CloudKit CKShare，苹果原生端到端加密。每笔交易三档可见性：家庭可见 / 仅伴侣 / 仅自己。数据永远不离开 Apple 生态。 不用自建同步服务器，家庭成员间通过 iCloud 账号直连。完美契合零云端约束。
④ 财务健康与「防御性」记账
沉没成本分析 V1.5 采纳 基于订阅管理延伸：标记「30/90 天未使用」的订阅 + 一次性购买但未被复用的 App/硬件/课程。AI 给出「建议取消」清单。 解决极客人群特有的囤货焦虑，差异化 + 自省价值。
情绪记账 V1.0 采纳 · 轻量 记账页新增可选「心情」chips：开心 / 后悔 / 刚需 / 犒赏 / 焦虑。年度回顾可做「快乐花费 vs 后悔花费」。 开发成本低但让产品有温度，配合年度报告形成情感钩子。
深圳生活成本基准（匿名社区） — 删除 必然需要汇总服务器；与「零云端」原则直接冲突。永久移除。 本地优先架构不适合做聚合社区类功能。
iCloud 同步 V1.0 采纳 · 架构基础 CoreData + NSPersistentCloudKitContainer。用户 iPhone / iPad / Mac 三端数据自动同步，苹果提供端到端加密。首次登录用户 iCloud 即可，无需注册账号。 零云端方案的同步基石，用户不用信任任何第三方，只信任苹果。
